| 编号 | 层级 | 自动化 | 通过标准 |
|---|---|---:|---|
| INS-001 | ZIP | 是 | SHA256 与伴随文件一致，文件名含版本号 |
| INS-002 | EXE | 是 | `ProductVersion` 与候选版本完全一致 |
| INS-003 | 清单 | 是 | 无 workspace、ODB/CAE、`.env`、测试目录或疑似密钥 |
| INS-004 | 路径 | 是 | 中文与空格路径下可启动、训练自检并退出 |
| INS-005 | 窗口运行库 | 是 | 解压后的 EXE 能加载 pywebview 与 WebView2 原生程序集 |
| CFG-001 | 离线 | 是 | 外部代理不可达时核心烟测仍通过 |
| CFG-002 | 升级 | 是 | 已有用户 workspace 文件不被覆盖 |
| CFG-003 | 错误 | 是 | 端口占用快速失败，日志包含可执行原因 |
| MCP-001 | 协议 | 是 | ping、execute、Unicode、模型和 Job 返回可读 |
| MCP-002 | 故障 | 是 | 超时、非法 JSON、错 ID、内核异常均不误报成功 |
| ABQ-005 | 实时 | 是/实机 | 当前 CAE 模型与 Job 上下文可读取，不只有心跳 |
| PH-001 | 建模 | 是/实机 | 生成 CAE、INP 和确定性配置 |
| PH-002 | 求解 | 是/实机 | Job 完成且 ODB、STA 非空 |
| PH-003 | 后处理 | 是/实机 | 从 ODB 提取位移、反力、Mises、PEEQ 和孔边特征 |
| PH-004 | 工程校核 | 是/实机 | 位移、反力、应力集中比落入 fixture 容差 |
| PH-005 | 案例库 | 是/实机 | 状态为 `archived` 且存在 `case_id` |
| UI-001 | 源码 UI | 是 | 系统诊断与带孔板页面可导航，关键控件可见 |
| UI-002 | 冻结 UI | 是 | ZIP 内 EXE 的私有服务通过同一 UI 流程 |
| CASE-001 | 案例治理 | 是 | 案例库显示显式单位制与“仅导出质量门合格案例”开关 |
| PHB-001 | 批量安全 | 是 | 带孔板批量页可创建计划，未确认时“运行真实求解”保持禁用 |
